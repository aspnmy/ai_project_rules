# db_config 目录说明

## 目录功能
此目录用于存储数据库配置相关的JSON配置文件，包含数据库操作、备份策略等配置信息。

## 配置文件列表
- **backup_strategy.json**: 数据库备份策略配置
- **file_operations.json**: 文件操作相关配置
- **maintenance_plan.json**: 数据库维护计划配置
- **mcp_config.json**: MCP服务器配置文件

## 配置文件说明
- 所有配置文件均采用JSON格式
- 配置文件应根据实际环境进行相应调整
- 配置更改应遵循项目变更管理流程

## 注意事项
- 敏感信息不应直接存储在配置文件中
- 配置文件应定期备份
- 修改配置后应重启相关服务使配置生效
