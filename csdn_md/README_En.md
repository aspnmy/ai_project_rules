# csdn_md Directory Description

## Directory Function
This directory is used to store project-related technical blog Markdown files, primarily for technical articles published on platforms like CSDN.

## File Types
- Technical blog articles
- Project usage tutorials
- Technical sharing documents
- Experience summary articles

## File Naming Rules
- Document file naming format: csdn_{FunctionName}_使用指南.md
- The function name should clearly describe the content theme covered by the article
- Ensure that blog content is kept updated in sync with project functions

## Document Management
- Technical blog content should be appropriately adjusted based on official documents in the docs directory
- Blog style should comply with reading habits of platforms like CSDN
- Important updates should be simultaneously published to technical blogs
- Regularly check the accuracy and timeliness of blog content
