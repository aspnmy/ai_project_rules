# 开发项目私有规范配置目录

## 目录说明

本目录用于存放开发项目的私有规则配置文件，包含以下文件：

- `myproject_rules.txt` - 私有规则定义文件
- `myproject_vars.txt` - 私有变量定义文件

## 文件功能说明

### myproject_rules.txt

私有规则定义文件，用于存放项目特定的规则配置，包括：
- 项目路径变量定义
- 规范类型设置
- 私有规则条目定义

### myproject_vars.txt

私有变量定义文件，用于存放项目特定的变量，所有变量以`My_`前缀命名。

## 使用方法

1. 编辑`myproject_rules.txt`添加或修改项目规则
2. 编辑`myproject_vars.txt`添加或修改项目变量
3. 规则文件需要在`rules\project_vars.txt`中注册才能生效

## 规范要求

- 规则文件必须使用相对路径，禁止使用绝对路径
- 变量命名必须符合`My_`前缀规范
- 修改规则后需要重启相关服务使配置生效
