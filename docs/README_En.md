# docs Directory Description

## Directory Function
This directory is used to store function usage guides and documentation for the project, including usage instructions and operation guides for various functional modules.

## Document Types
- Function usage guides
- Operation manuals
- Configuration instructions
- Best practice documentation

## Document Naming Rules
- Document file naming format: docs_{FunctionName}_使用指南.md
- The function name should clearly describe the functional module covered by the document
- Ensure that document content is kept updated in sync with function implementation

## Document Management
- When adding new functional modules, corresponding usage guides should be created simultaneously
- Document content should be regularly reviewed and updated
- Ensure the accuracy and completeness of documents
- Important changes should be clearly marked in the documentation
