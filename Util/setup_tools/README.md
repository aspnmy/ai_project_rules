# setup_tools 目录说明

## 目录功能
此目录包含项目设置和配置相关的工具脚本，主要用于服务器安装、开发环境配置等功能。

## 工具列表
- **install_mcp_servers.py**: MCP服务器安装工具
- **install_podman_windows.py**: Windows环境下Podman安装工具
- **set_SystemPathVar.py**: 系统路径变量设置工具

## 功能说明
- 自动化安装MCP服务器
- 在Windows环境中安装和配置Podman
- 设置和管理系统环境变量
- 帮助用户快速搭建开发和运行环境

## 使用方法
- 根据需要运行相应的安装和配置工具
- 确保以管理员权限运行需要系统级修改的工具
- 按照工具提示完成安装和配置过程
- 安装完成后，验证安装结果以确保环境正确设置
