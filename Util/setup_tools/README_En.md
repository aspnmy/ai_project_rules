# setup_tools Directory Description

## Directory Function
This directory contains project setup and configuration-related tool scripts, primarily used for server installation, development environment configuration, and other functions.

## Tool List
- **install_mcp_servers.py**: MCP server installation tool
- **install_podman_windows.py**: Podman installation tool for Windows environment
- **set_SystemPathVar.py**: System path variable setting tool

## Function Description
- Automatically install MCP servers
- Install and configure Podman in Windows environment
- Set up and manage system environment variables
- Help users quickly set up development and runtime environments

## Usage
- Run the corresponding installation and configuration tools as needed
- Ensure administrator privileges when running tools that require system-level modifications
- Follow the tool prompts to complete the installation and configuration process
- After installation, verify the installation results to ensure the environment is correctly set up
