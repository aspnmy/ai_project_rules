# check_tools Directory Description

## Directory Function
This directory contains tool scripts for checking project configurations, rules, and gateway domains, primarily used to verify the compliance and correctness of the project.

## Tool List
- **check_gateway_domains.py**: Tool for checking gateway domain configurations
- **check_project_config.py**: Tool for checking project configurations
- **check_rules_manager.py**: Tool for checking rules manager

## Tool Function Description
- Verify that configuration files are formatted and content compliant with specifications
- Check the validity of domain and gateway configurations
- Ensure the rules manager works properly
- Provide diagnostics and repair suggestions for configuration issues

## Usage
- Run the corresponding check tool script as needed
- Scripts typically output check results and potential issues
- Make appropriate adjustments and repairs based on the output information
