# check_tools 目录说明

## 目录功能
此目录包含用于检查项目配置、规则和网关域等的工具脚本，主要用于验证项目的合规性和正确性。

## 工具列表
- **check_gateway_domains.py**: 检查网关域名配置的工具
- **check_project_config.py**: 检查项目配置的工具
- **check_rules_manager.py**: 检查规则管理器的工具

## 工具功能说明
- 验证配置文件的格式和内容是否符合规范
- 检查域名和网关配置的有效性
- 确保规则管理器正常工作
- 提供配置问题的诊断和修复建议

## 使用方法
- 根据需要运行相应的检查工具脚本
- 脚本通常会输出检查结果和可能的问题
- 根据输出信息进行相应的调整和修复
