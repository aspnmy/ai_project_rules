# WSL Configuration Files Directory

This directory is used to store configuration files related to Windows Subsystem for Linux (WSL).

## Included Files

- **wsl_config.json**: Main WSL configuration file
- **wsl-distro.info**: WSL distribution information
- **wsl_dev_manager.sh**: WSL development environment management script

## Usage Instructions

Store all WSL-related configuration files in this directory for unified management and maintenance.
